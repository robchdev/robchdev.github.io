Come installare

  1. Scarica l'APK con il pulsante "Scarica APK".
  2. Apri il file scaricato (`app-release.apk`).
  3. Se richiesto, abilita "Installa app sconosciute" per browser/file manager.
  4. Conferma con "Installa" e attendi il completamento.
  5. Apri Easter Eggs Maker dal launcher.

Se Android blocca l'installazione

  • Vai in Impostazioni > Sicurezza/Privacy.
  • Abilita temporaneamente l'installazione da origine sconosciuta per l'app che stai usando.
  • Dopo l'installazione puoi disattivare di nuovo il permesso.
